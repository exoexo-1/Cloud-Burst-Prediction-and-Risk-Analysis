# HydroPrognosis вЂ“ Cloudburst Prediction and Flood Risk Analysis

> Real-time flood vulnerability assessment and cloudburst prediction for Uttarakhand, powered by fuzzy logic, machine learning, and RAG-augmented LLM analysis.

**Live:** https://hydroprognosis.vercel.app

---

## What It Does

HydroPrognosis lets anyone click a point on a map and instantly get:

- A **Flood Vulnerability Index (FVI)** score (0вЂ“100) computed in real time using fuzzy logic
- A **cloudburst probability** (%) predicted by a trained ML model
- A full **risk analysis report** generated by GPT-4o-mini, grounded in a district-level knowledge base via RAG
- A **72-hour forecast** with safety recommendations for residents and authorities

The system is designed for Uttarakhand вЂ” a region that experiences frequent and deadly cloudbursts вЂ” and covers all 13 districts with district-specific knowledge bases.

## How FVI Is Computed

The **Flood Vulnerability Index** is calculated using a fuzzy logic system with 6 input variables:

| Input | Source |
|---|---|
| Rainfall (mm) | Open-Meteo live forecast |
| Slope (degrees) | Open-Elevation (sampled nearby points) |
| Imperviousness (%) | Derived from urbanization + socioeconomic data |
| Distance to water (m) | OSMnx waterway/water feature query |
| Soil moisture | Open-Meteo hourly data |
| Elevation (m) | Open-Elevation API |

Membership functions and fuzzy rules map these inputs to an FVI score (0вЂ“100), which is then categorised as Very Low / Low / Moderate / High / Very High. A fallback linear scoring function handles cases where the fuzzy system cannot converge.

---

## How Cloudburst Probability Is Predicted

A pre-trained regression model (`cloudburst_regressor.pkl`) predicts cloudburst probability from a **14-feature vector** built from live and historical weather data:

- Min/max temperature, humidity, pressure
- Precipitation sum, rain sum, precipitation probability
- Cloud cover, wind speed, wind gusts, wind direction (cardinal encoded)
- Is-day flag, temperature at 2m, weather code (encoded)

Previous week averages are fetched from Open-Meteo Archive API and blended with current conditions to capture antecedent moisture вЂ” a key cloudburst risk factor.

## RAG Knowledge Base

The `Knowledge Base/` directory contains structured markdown documents for each of Uttarakhand's 13 districts, covering:

- Risk factors and vulnerability profiles
- Historical cloudburst and flood events (1970вЂ“2025)
- Topography, geology, and hydrology
- Microclimate and terrain-specific patterns

These documents are embedded using OpenAI Embeddings and stored in ChromaDB. At query time, 4 targeted queries per district retrieve the most relevant context, which is passed to GPT-4o-mini alongside the FVI data and ML prediction.

## API Reference

### `GET /fvi`
Computes Flood Vulnerability Index for a coordinate.

**Query params:** `lat`, `lon`, `district` (optional)

**Response:**
```json
{
  "fvi_score": 62.4,
  "risk_level": "Moderate",
  "key_factors": ["Moderate rainfall (34.2 mm)", "Close to water body (1200 m)"],
  "inputs": { "weather": {...}, "terrain": {...}, "hydrology": {...} }
}
```

### `POST /analysis`
Generates a full risk report using RAG + ML + GPT.

**Body:**
```json
{
  "place_name": "Rudraprayag",
  "fvi_data": { ... }
}
```

**Response:**
```json
{
  "analysis": "### 1. Flood Risk Level\nHigh...",
  "rag_context": "CONTEXT FOR RUDRAPRAYAG:...",
  "status": "success"
}
```
